GIS-INTRO-RCAAS_RSE

Documentation: https://university-of-glasgow-public.github.io/gis-intro-rcaas-rse/

Web application using Flask and Python to demonstrate the use of the Leaflet JavaScript Library and the document-oriented NoSQL MongoDB database system to build maps depicting points (using Scottish bothy locations as an example) and polygons (using national parks as an example). One of the aims was to explore the synergy between using GeoJSON within Leaflet and a JSON-like database system such as MongoDB to store those data; to live in the JSON world. Examples of how to define and create polygons using the map interface and how to perform geospatial searches for points within the polygons or within a circle of specified radius are illustrated. The application also aims to illustrate how to choose and add license information and how to add copyright information.
